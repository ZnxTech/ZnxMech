# ZnxMech

.env file structure:

```
// client settings
CLIENT_ID = *client id*
CLIENT_SECRET = *client secret*

// client owner settings
OWNER_ID = *client owner user id*
OWNER_NICK = *client owner login name*

// bot settings
BOT_ID = *bot user id*
BOT_NICK = *bot login name*
BOT_OAUTH = *bot oauth from some generator*
```
